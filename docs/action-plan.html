Run both at once

Two tracks

One engine earns while the other is built. Track B never spends ahead of Track A's proof.

Track A — Revenue now

Rev-1 kits + restaurant pilots

The cash engine. No FCC gate, near-zero capital. Validates the exact same demand as the future assembled unit while it gets designed.

  • Pre-flashed DIY kits on Tindie (buyer completes the radio)
  • Free loaner pilots → deposits, paid installs, certified mini-PC
  • Free local-first hub does the selling: "it just showed up on my dashboard"
Gate: none — kits & loaners aren't a finished-radio sale.
Track B — Scale later

Rev-2 module + FCC SDoC

Respin around the pre-certified ESP32-C3-MINI-1 so a cheap Part 15B SDoC (~$300–1,500) replaces $5k–15k bare-chip testing. Firmware is unchanged.

  • Design the KiCad schematic + PCB now — it costs $0
  • Export Gerber/BOM/CPL, then park it
  • Order PCBA + run the SDoC only once demand is real
Gate: proven demand — pilots convert or kits sell steadily.

Sequence

The timeline

Front-load everything free; gate the two real spends — the PCBA batch and the FCC SDoC — behind revenue you've already collected.

  1. This weekno FCC · ~$100–700 · mostly your time
    • Form Datum Labs LLC (your liability shield + future FCC responsible party), free EIN at IRS.gov, business bank account.
    • Turn on no-inventory revenue: Stripe pre-order link, Ko-fi + Sponsors, Gumroad PWYW hub + paid setup call.
    • Edit the warranty with real names + the liability cap; ship it with every kit and pilot. Pull 2–3 insurance quotes.
    • Order kit parts for 10, set up the flashing PC + bench hub, and text your warmest restaurant contact.
    • Start the rev-2 KiCad design — free, do it while the boards ship.
  2. Weeks 2–4boards land
    • Incoming QC → validate the toolchain on one board (the gate) → build a no-solder test rig.
    • Pair-and-test the batch; shoot photos; publish the kit-only Tindie listing.
    • Build 2–3 loaners; run 15-min qualifying calls; install the first pilot and do the silent-sensor demo live.
    • Bind insurance before hardware enters a commercial kitchen. Finish + export rev-2, then park it.
  3. Month 2–3
    • Fulfill kit orders in 3–5 days; carry a few to a local meetup for cash.
    • Run pilots 30 days; capture every real event it catches. Convert (day 25–30) → deposit + referral + testimonial.
    • If demand is real: order the rev-2 PCBA batch (5–10 units, ~$150–350) and bring them up.
  4. The gate — when demand is provenonly now spend on certification
    • Run the FCC Part 15B SDoC (~$300–1,500) on a QC-passed rev-2 unit; keep the signed report.
    • Label "Contains FCC ID: 2AC7Z-ESPC3MINI1"; confirm insurance bound; COI + W-9 ready.
    • Deliver certified units to reserved pilots; sell assembled B2B + a homelab variant on Tindie.
    • Only now consider financing — for inventory scaling, never certification.

Hold the honest disclaimer everywhere: ±0.5 °C typical, uncalibrated; a loss-prevention & logging aid, not a certified, NIST-traceable, or HACCP compliance instrument; it doesn't replace required manual checks, and monitoring stops in a full power outage. Never claim a certification you don't hold.
